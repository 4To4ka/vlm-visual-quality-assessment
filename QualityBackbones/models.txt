Canonical extraction rules and original-style snippets

1) ResNet and ResNeXt CNN families
Method:
- Per-layer: global average pooling over spatial dimensions for feature maps.
- Canonical embedding: model pooled output for HF ResNet; stage feature-map GAP for timm feature extractors.
Snippet:
```python
# x: [B, C, H, W]
emb = x.mean(dim=(-2, -1))
```

2) ConvNeXt
Method:
- Per-layer: global average pooling over spatial maps.
- Canonical embedding: LayerNorm(global_avg(last_hidden_state)) via HF pooler_output.
Snippet:
```python
# x: [B, C, H, W]
emb = layernorm(x.mean(dim=(-2, -1)))
```

3) ViT, DINOv1, DINOv2, MAE, InternViT
Method:
- Per-layer: CLS/first-token pooling for token sequences.
- Canonical embedding:
  - DINOv2: pooler_output (post-layernorm CLS).
  - ViT/DINO/MAE/InternViT: first token from final hidden state path.
Snippet:
```python
# tokens: [B, N, D]
emb = tokens[:, 0]
```

4) DINOv3 (timm Eva models with prefix tokens)
Method:
- Per-layer: average over patch tokens, excluding prefix tokens.
- Canonical embedding: model.forward_head(model.forward_features(x), pre_logits=True).
Snippet:
```python
# tokens: [B, N, D], num_prefix_tokens = p
emb = tokens[:, p:, :].mean(dim=1)
```

5) I-JEPA
Method:
- Per-layer: mean over all patch tokens (no CLS token in embeddings path).
- Canonical embedding: mean of final token sequence (HF path aligned to patch-token semantics).
Snippet:
```python
# patch_tokens: [B, N, D]
emb = patch_tokens.mean(dim=1)
```

6) CLIP vision
Method:
- Per-layer: CLS/first-token pooling from hidden states.
- Canonical outputs exported:
  - pooler_output (post-layernorm CLS)
  - image_embeds (visual projection)
  - image_embeds_l2 (L2-normalized projected embedding)
Snippet:
```python
pool = post_layernorm(tokens[:, 0])
img = visual_projection(pool)
img = img / img.norm(dim=-1, keepdim=True)
```

7) SigLIP and SigLIP2 vision
Method:
- Per-layer: post_layernorm + attention pooling head (model head) when available.
- Canonical outputs exported:
  - pooler_output (attention-pooled)
  - image_embeds_l2 (L2-normalized pooler output)
Snippet:
```python
tokens_ln = post_layernorm(tokens)
pool = attn_pool_head(tokens_ln)
img = pool / pool.norm(dim=-1, keepdim=True)
```

8) ARNIQA
Method:
- Canonical embedding: repository return_embedding output (concatenated full-scale + half-scale path).
Snippet:
```python
score, emb = model(full_res, half_res, return_embedding=True)
```

9) MANIQA
Method:
- Canonical project embedding path: internal token features reconstructed then mean pooled over patches.
Snippet:
```python
# h_tokens: [B, N, D]
emb = h_tokens.mean(dim=1)
```
